# ペン回し練習室

## サービス概要

ペン回しのコンボ、フリースタイルをメモするツール

- コンボ…複数のペン回しの技を組み合わせたもの
- フリースタイル…複数の技・コンボを組み合わせた一連の流れ
  - ペン回し愛好家の間で主流の楽しみ方

## 使い方

#### ペン回しのコンボ・フリースタイルをメモ

- 練習したいコンボ・フリースタイルを構成する技をメモ
- 技は編集・削除・ドラッグ＆ドロップによる並び替えを行うことが可能

#### ログイン時

- 複数のコンボ・フリースタイルを登録することが可能になる
- コンボ・コンボ名は編集、削除が可能
